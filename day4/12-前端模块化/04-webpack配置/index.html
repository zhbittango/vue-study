1、使用node前需要初始化npm init：生成package.json，里面有scripts脚本可以代替简化执行命令

2、webpack.config.js：配置entry、output 直接使用webpack打包即可，或者使用替代命令npm run bulid

3、安装局部webpack：因为项目可能使用的webpack版本不一样，所以需要下载项目相对应webpack版本 npm install webpack@3.6.0 --save-dev;

4、注：终端使用webpack命令用的都是全局的版本；解决方法：使用package.json的scripts脚本代替webpack命令： npm run build; 先从局部查找webpack版本，后找全局
