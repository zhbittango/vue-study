hello webpack

搭建本地服务器

1、安装npm install --save-dev webpack-dev-server@2.9.0
2、添加script脚本
3、配置webpack.config.js

loader

css
1、新建css文件，在main中require引用
2、安装css-loader、style-loader模块依赖,配置webpack.config.js文件

less 安装less-loader模块依赖,配置webpack.config.js文件

img 在css中使用背景图url时需要安装url-loader模块依赖,配置webpack.config.js文件;安装file-loader模块依赖

es6转es5 npm install --save-dev babel-loader@7 babel-core babel-preset-es2015,配置webpack.config.js文件

使用vue npm install --save vue,配置webpack.config.js文件

需要安装开发时依赖 npm install --save-dev vue-loader vue-template-compiler,配置webpack.config.js文件
